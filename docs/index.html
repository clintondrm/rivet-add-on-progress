Process Step Indicator

Displays each step in an ordered sequence of steps, indicating the current step and allowing users to jump to any step. Both horizontal and vertical variants available.

Default (Horizontal) Configuration

By default, items are shown horizontally. Be sure to allow enough room for the listed items on expected device and container widths.

  2. Step 2 2

    ...
</ol>

When to use

  • To communicate how many steps there are in a structured process
  • To communicate completion of required steps in a structured process

When to consider something else

  • For general navigation, use a dedicated nav element
  • Navigation for situations when users do not need to complete all steps of a process
  • For normal lists, use ordered lists or unordered lists.

Accessibility

Relies on standard interactive element (anchor tags) to enable keyboard-only navigation. Accessible implementation will require the use of descriptive labels within the anchor tags.
